Toast is a Python library with a collection of general-use functions that are commonly used in the development of algorithms for the Cybernetic Implantable Devices Lab (CID). These functions are useful in performing a variety of tasks relevant to machine learning including data segmentation, labeling and processing. Also included are frameworks for model fitting, feature generation, model selection, model evaluation, and other numerical computations.

The library is designed to be highly modular and easy to use, making it a useful tool for both researchers and engineers working in the field of machine learning. Overall, the toast library aims to provide a set of powerful and flexible tools for developing cutting-edge machine learning models.
